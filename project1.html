Project Overview

This project was created for the University of Canberras 24-hour Design Challenge with the theme "Lost and Found". The challenge was to create a digital design that explored the concept of being lost and found.

The Issues

24 Hours to completely conceptualise, design, code and deploy a digital design, in my case, an interactive narrative. Specific to my project, issues included:

  • Devising a narrative that used the theme effectively, requiring quick storyboarding and ideation
  • Integrating complex scroll interactions required limiting constraints
  • Creating a seamless user experience that conveyed my use of the theme 404 errors as a metaphor

My Solution

My solution focused on creating an immersive experience using GSAP that guides a user from 404 (lost) to 302 (found) through interactive storytelling and smooth animations.

  • Concept Ideation: Explored digital disconnection, looking at abstractions of being 'lost' online and the emotional journey towards being 'found.'
  • Design Choices: CMD Console aesthetic to create a sense of familiarity and nostalgia
  • Technical Implementation: Used GSAP (Green Sock Animation Platform) to implement scroll-triggered animations and click interactions to reveal story elements progressively. Making for an intuitive UI.

Key Features

  • Narrative structure that uses interactivity constraints to ensure intended user progression, ie users can only proceed once clicking or answering intended prompts.
  • Use of GSAP for multiple animations that enhance storytelling and atmosphere, ie split text, glitch effects and smooth scrolling.
  • Created in under 24 hours.
