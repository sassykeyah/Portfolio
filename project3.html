Project Overview

A commission for the Parliamentary Education Office of Australia. The goal was to create tactile resources that showcase the layout of the two chambers for individuals who are vision impaired

The Challenge

My role was transforming a coloured, complex layout into a simplistic tactile map that conveys the information through tactile elements.

  • Simplifying the original designs to feature different uses of ink stippling.
  • Understanding how the PIF (Print in a Flash) machines work to create tactile elements.
  • Ensuring the final product was accessible and met user needs.

Design Process

Detail your approach and methodology:

  • Research: Conducted analysis into different tactile mapping techniques and representations.
  • Ideation: Defined element hierarchy and user flows. Tested different layouts with the braille numbering system.
  • Design: Created high contrasting, block elements to represent difference in elements. For example block benches and stippled tables.
  • Development: Production of tactile maps using PIF machines.

Key Features

  • High contrasting elements.
  • Tactile representations of key architectural and layout features.
  • Integration of braille labels for navigation.
