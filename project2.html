Project Overview

Spotify Rewind is an interactive web application targeted at music enthusiasts and the general population who are interested in the connections between music and Australia's history. It aims to provide an engaging way to explore significant historical events the highest charting ARIA songs from the corrosponding years.

The Challenge

A highly technical project, it required addressing these challenges:

  • Successful integration of Spotify's API using Vue.js and cURL
  • Understand the target user and what they would like to gain from the experience.
  • Depict over 15 key events in Australia’s history with corresponding ARIA top song of the year.
  • Ensure web application is responsive to all device sizes.

My Solution

My solution focused on creating an interactive, horizontal timeline that allows the user to scroll and click on each event:

  • Concept Ideation: Explored the NMAS timeline of important events in Australia's history and correlated them with the top ARIA songs from those years.
  • Design Choices: Retro design choices to envoke a sense of nostalgia, use of an animating backrground (referencing all those cool 80's shapes we all love!)
  • Technical Implementation: Integration of cURL to authenticate and fetch data from Spotify's API, and Vue.js to build a dynamic and responsive user interface.

Key Features

  • Horizontal timeline, it allows users to scroll through Australia's history in a chronological manner and the relevant music.
  • Showcases key historical events alongside their corresponding top ARIA songs, providing context and enhancing user engagement. Implemented a 'details' page for further exploration.
  • Abilities to click a button to navigate directly to the spotify webplayer to listen to the selected songs.
